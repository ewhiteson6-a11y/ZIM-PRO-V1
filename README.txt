ZimDrive Pro - Full PWA package

Deploy on Vercel:
1. Upload the contents of this folder or the zip to a Vercel project.
2. Framework preset: Other
3. Build command: leave blank
4. Output directory: leave blank

iPhone/iPad offline setup:
1. Open the deployed site in Safari.
2. Use Share -> Add to Home Screen.
3. Launch from the home screen at least once while online.
4. After that, the app shell stays cached for offline use.

Files included:
- index.html
- manifest.webmanifest
- service-worker.js
- icons/
- vercel.json
